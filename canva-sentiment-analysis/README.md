# Canva User Review Sentiment Analysis Using NLP

## Description
This project applies Natural Language Processing (NLP) techniques to **analyze sentiment** from user reviews of Canva. The analysis transforms unstructured textual data into structured numerical features to classify sentiment categories and extract meaningful insights.

## Dataset
The dataset consists of Canva user reviews scraped from Google Play Store, containing textual feedback that reflects user experiences and satisfaction levels.

## Project Objectives
1. Preprocess textual data using tokenization, stopword removal, and normalization techniques.
2. Convert text data into numerical representations using feature extraction methods.
3. Build classification models to predict user sentiment.
